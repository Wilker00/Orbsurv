Security Practices

Encryption

All data is encrypted in transit using TLS 1.3 and at rest using AES-256 encryption. Video feeds are encrypted end-to-end.

Access Control

Role-based access control with multi-factor authentication support. All access is logged and audited.

Security Testing

Regular penetration testing, vulnerability assessments, and security audits by third-party firms.

Data Privacy

On-device AI processing minimizes data transmission. Privacy zones and face blurring protect sensitive information.

Audit Reports

We conduct regular security audits and make reports available to enterprise customers under NDA.
