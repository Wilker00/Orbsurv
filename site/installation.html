Installation Guide

Step-by-step instructions to install and configure your Orbsurv Patrol Module. Most installations take 2-4 hours.

Prerequisites

Hardware Requirements

Ensure you have everything needed before starting installation

Power Requirements

  • 90W PoE++ (Power over Ethernet Plus Plus) switch or injector
  • Alternative: Dedicated 90W power adapter
  • Voltage: 48V DC
  • Backup battery recommended for critical deployments

Network Requirements

  • Gigabit Ethernet connection (PoE++)
  • Wi-Fi 6E router (if using wireless)
  • Optional: 5G modem for remote locations
  • Minimum 10 Mbps upload bandwidth

Installation Tools

  • Drill with appropriate bits for mounting surface
  • Level for rail alignment
  • Measuring tape
  • Stud finder (for wall mounting)
  • Included mounting hardware

Setup Device

  • iOS or Android device with Orbsurv app
  • Or web browser for web-based setup
  • Active internet connection
  • Orbsurv account (created during setup)

Safety First

Always ensure power is disconnected before handling electrical components. For ceiling installations, use appropriate safety equipment. If unsure about any step, consult a professional installer or contact Orbsurv support.

Installation

Step-by-Step Installation

Follow these steps in order for a successful installation

1

Unbox and Verify Components

Open your Orbsurv package and verify all components are present. You should have:

  • Orbsurv Patrol Module (camera unit)
  • Rail segments (quantity depends on your deployment)
  • Mounting brackets with screws
  • Power supply (if not using PoE++)
  • Network cable (if using wired connection)
  • Quick start guide and documentation

Check for any shipping damage. Contact support immediately if any components are missing or damaged.

2

Plan Your Installation Location

Determine the optimal location for your rail system:

  • Measure your space to determine rail length needed
  • Identify mounting surface (ceiling or wall)
  • Ensure rail path provides complete coverage
  • Check for power and network connectivity
  • Verify mounting surface can support weight (module + rail)
  • Consider cable routing and accessibility

Use the Orbsurv app's coverage planning tool to visualize your installation before mounting.

3

Install Mounting Brackets

Mount the brackets that will hold your rail system:

  • Mark bracket positions using a level and measuring tape
  • For ceiling: Locate studs or use appropriate anchors
  • For wall: Use stud finder to locate solid mounting points
  • Drill pilot holes at marked positions
  • Secure brackets with provided screws
  • Verify brackets are level and securely mounted

Bracket spacing should match your rail segment lengths. Typically 2 meters between brackets.

4

Assemble and Mount Rail Segments

Connect rail segments and mount to brackets:

  • Connect rail segments using snap-fit connectors
  • Ensure segments are aligned and flush
  • Mount rail assembly to brackets
  • Verify rail is level across entire length
  • Check for any gaps or misalignments
  • Route cables through hidden cable channel

The rail should be smooth and free of obstructions. Test by manually moving a small object along the rail path.

5

Mount Patrol Module

Attach the Patrol Module to the rail:

  • Position module at one end of the rail
  • Engage module's mounting mechanism with rail
  • Verify module moves freely along rail
  • Check that module doesn't wobble or bind
  • Ensure all connections are accessible

The module should slide smoothly without excessive force. If movement is rough, check rail alignment.

6

Connect Power and Network

Connect power and network cables:

  • Connect PoE++ cable to module and switch/injector
  • If using Wi-Fi: Configure network settings through app
  • If using 5G: Insert SIM card and activate
  • Verify power LED indicators show normal operation
  • Check network connectivity status
  • Ensure cables don't interfere with module movement

Allow 2-3 minutes for module to boot and initialize. Status LED will indicate when ready.

7

Download and Setup App

Set up the Orbsurv mobile or web app:

  • Download Orbsurv app from App Store or Google Play
  • Create an Orbsurv account or sign in
  • Follow in-app setup wizard
  • Scan QR code on module or enter device ID
  • Connect to your module's network
  • Complete initial account setup

The app will guide you through device pairing and network configuration step by step.

8

Initial Calibration

Calibrate your system for optimal performance:

  • Run automatic calibration from app
  • System maps your space and rail path
  • Define patrol boundaries
  • Set initial waypoints
  • Configure detection zones
  • Test module movement along full rail length

Calibration typically takes 15-30 minutes. The system learns your space over the first few days to improve accuracy.

9

Configure Settings and Preferences

Customize your Orbsurv system:

  • Set up patrol schedules for different times
  • Configure detection sensitivity
  • Define privacy zones if needed
  • Set up alert preferences and notifications
  • Configure storage and cloud sync options
  • Test motion detection and alerts

These settings can be adjusted anytime through the app. Start with defaults and fine-tune as needed.

Next Steps

After Installation

Important steps to complete your setup

Verify Coverage

Test the module's movement along the entire rail. Use the app to manually position the module and verify it reaches all areas of your space without blind spots.

Let AI Learn

Allow 2-3 days for the system to learn your space. The AI will adapt to normal patterns, reducing false alarms and improving detection accuracy over time.

Fine-Tune Settings

Adjust sensitivity, alert preferences, and patrol schedules based on your needs. Review detection events and refine zones to minimize false alarms.
